# LDAP Console

A professional React, Tailwind CSS, and shadcn-style dashboard starter for an LDAP identity operations product.

## Features

- Vite + React + TypeScript application scaffold.
- Tailwind CSS theme using shadcn-compatible CSS variables.
- Polished login experience with an enterprise identity palette.
- Responsive dashboard for LDAP metrics, replication status, connectors, and audit activity.

## Getting started

```bash
npm install
npm run dev
```

## Available scripts

- `npm run dev` starts the local Vite development server.
- `npm run build` type-checks and builds the production bundle.
- `npm run lint` runs ESLint against the source tree.
